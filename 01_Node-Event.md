# Node Event

## 1. 개요
Fair Duel의 모든 방(Node) 이벤트에 대한 문서.
우리 게임은 로그라이크 게임으로써 **노드의 다양성** 이 반드시 확보되어야 한다.
따라서 이곳의 정리된 노드들은 모두 다른 경험을 주도록 디자인 되었으며, 각 공간은 고유한 보상/리스크/내러티브 를 가진다.

## 2. 노드 이벤트 종류
- [[01Choice-Room]] — 플레이버 텍스트를 토대로 주어진 선택지를 통해 얻고 잃는 공간
- [[01Shop]] — 증표를 소모해 덱을 가다듬는 방
- [[01Battle-Room]] — 층 카드팩을 보상으로 주는 Nomal-Enemy전투방
- [[01Elite-Battle-Room]] — 카드군의 핵심 코어 카드를 얻을 수 있는 도전 전투방
- [[01Altar]] — 보스 버프를 완화하는(대가가 따르는) 특수방
