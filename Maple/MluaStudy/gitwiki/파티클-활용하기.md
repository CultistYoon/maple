파티클 프로퍼티 제어

파티클이 잘 보이도록 어두운 배경을 설정하고 지형을 배치한 뒤 원하는 파티클을 배치합니다. 본 예제에서는 Background로 back-16을 설정하고 파티클은 Nova를 배치했습니다.

workspace Workspace - MyDesk MyDesk에 새 스크립트 컴포넌트 scriptcomponent ParticlePlaySpeed를 생성합니다.

맵에 배치했던 파티클 엔티티의 property Property에 etc ParticlePlaySpeed를 추가합니다.

scriptcomponent ParticlePlaySpeed를 열고 OnUpdate() 함수를 추가하여 아래와 같이 스크립트를 작성합니다.

[client only]
void OnUpdate(number delta)
{
    local positionA = Vector2(self.Entity.TransformComponent.Position.x,self.Entity.TransformComponent.Position.y)
    local positionB = Vector2(_UserService.LocalPlayer.TransformComponent.Position.x,_UserService.LocalPlayer.TransformComponent.Position.y)
    local distance = Vector2.Distance(positionA, positionB)

    self.Entity.BasicParticleComponent.PlaySpeed = distance
}

scriptcomponent ParticleServiceTest를 열고 AddParticle() 함수를 추가한 뒤 아래와 같이 작성합니다.

void AddParticle()
{
    local option = {["Color"] = Color(0.25,0.5,0.5, 1), ["ParticleCount"] = 3}

    _ParticleService:PlayBasicParticle(BasicParticleType.Firework, self.Entity, self.Entity.TransformComponent.WorldPosition, 0, Vector3(1,1,1), false, option)
    -- ParticleService의 PlayBasicParticle() 함수를 통해 파티클을 재생합니다. Loop는 false로 설정하여 파티클이 한 번씩만 재생되도록 합니다.
}
OnBeginPlay() 함수를 추가한 뒤 아래와 같이 작성합니다.

[server only]
void OnBeginPlay()
{
    _TimerService:SetTimerRepeat(self.AddParticle, 2, 0) 
    -- 2초에 한 번씩 AddParticle 함수가 작동하도록 합니다.
}

반복 파티클의 생성과 삭제
반복 재생되는 파티클을 생성하거나 삭제하는 방법을 알아봅시다.
특정 엔티티에 캐릭터가 닿으면 파티클이 생성되다가 캐릭터가 목표 지점에 도착하면 파티클 생성이 멈추는 예제를 만들어보겠습니다.

파티클을 뿜어낼 엔티티와 목표 엔티티를 배치합니다. 본 예제에서는 파티클을 뿜어낼 엔티티로 object-24(호리병)을, 목표 엔티티로는 object-928(소화기)를 사용했습니다.

workspace Workspace - MyDesk MyDesk에 새 스크립트 컴포넌트 scriptcomponent ParticleServiceTest02를 생성합니다.

scriptcomponent ParticleServiceTest02를 열고 아래와 같이 프로퍼티를 추가합니다.

Property:
[None]
Entity ResetEntity = "nil"    
[None]
SyncTable<number> SpawnList
캐릭터가 호리병에 닿으면 호리병에서 파티클이 생성되고, 소화기에 닿으면 파티클이 사라지도록 해봅시다. scriptcomponent ParticleServiceTest02의 Event Handler에 event TriggerEnterEvent를 추가한 뒤 아래와 같이 작성합니다.

Event Handler:
HandleTriggerEnterEvent(TriggerEnterEvent event)
{
    -- Parameters
    local TriggerBodyEntity = event.TriggerBodyEntity
    --------------------------------------------------------
    if TriggerBodyEntity ~= self.ResetEntity then
       local serial = _ParticleService:PlayBasicParticleAttached(BasicParticleType.ToonTallFire, TriggerBodyEntity, Vector3.zero, 0, Vector3.one, true)
       table.insert(self.SpawnList, serial)
       -- 캐릭터가 목표 엔티티가 아닌 다른 엔티티에 닿으면 PlayBasicParticleAttached() 함수를 통해 해당 엔티티에서 파티클이 생성됩니다.
    else
       for idx, serial in pairs(self.SpawnList) do
           _ParticleService:RemoveParticle(serial)
       -- 캐릭터가 목표 엔티티에 닿으면 RemoveParticle() 함수를 통해 기존에 생성된 모든 파티클이 제거됩니다.
       end
       table.clear(self.SpawnList)

    end
}

Event Handler:
[self]
HandleButtonClickEvent(ButtonClickEvent event)
{
     --------------- Native Event Sender Info ----------------
     -- Sender: ButtonComponent
     -- Space: Client
     ---------------------------------------------------------

     -- Parameters
     -- local Entity = event.Entity
     ---------------------------------------------------------
     local Entity = event.Entity
     ---------------------------------------------------------
     if self.particleEntity.UIBasicParticleComponent.IsEmitting == true then
         self.particleEntity.UIBasicParticleComponent:Stop()
     end
         self.particleEntity.UIBasicParticleComponent:Play()
}
