리치 텍스트
리치 텍스트에서 제공하는 다양한 태그를 사용해 글씨를 강조하거나, 편집할 수 있습니다. 메이플스토리 월드에서 리치 텍스트를 사용할 수 있는 컴포넌트는 총 4가지로, TextRendererComponent, TextGUIRendererComponent, ChatBalloonComponent, TextComponent입니다. 이 중 TextComponent에서는 일부 리치 텍스트 태그만 사용할 수 있습니다.

리치 텍스트 사용
TextRendererComponent, TextGUIRendererComponent, ChatBalloonComponent에서 사용할 수 있는 태그 목록입니다. IsRichText 프로퍼티를 활성화해야 리치 텍스트를 사용할 수 있습니다.

<태그이름="값">

여러 개의 태그를 사용할 때는 </태그이름>을 사용해 태그가 적용되는 범위를 제한할 수 있습니다. 예를 들어 <mark=#FFFF0080><s>Maple</mark><color=#fa7bbf><cspace=10>Story</color><allcaps>World</allcaps>로 입력하면 아래와 같이 지정한 범위에서만 태그가 적용됩니다.

[object Object]

리치 텍스트 태그 목록
태그	설명	예시
align	정렬을 변경합니다.	<align="center">
allcaps	영어 알파벳을 모두 대문자로 변환합니다.	<allcaps>
alpha	불투명도를 변경합니다.	<alpha=#80>
b	굵은 글씨로 나타냅니다.	<b>글자 입력</b>
br	문장을 줄 바꿈합니다.	첫 번째 문장<br>두 번째 문장
color	색상을 결정합니다. 색상은 헥스 코드(#00000000) 형식으로 입력합니다. 색상명으로 색을 표현할 수 있는 이름은 아래와 같습니다.
black, blue, green, orange, purple, red, white, yellow
<color=#FF0000>,<color=green>
cspace	문자의 자간을 조정합니다.	<cspace=10>
i	기울임체로 변경합니다.	<i>
indent	문단을 내어쓰기 합니다.	<indent=20%>
line-height	줄 간격을 조정합니다.	<line-height=150%>
line-indent	문단의 첫 번째 줄을 들여쓰기 합니다.	<line-indent=20%>
lowercase	소문자로 변환합니다.	<lowercase>
margin	여백을 부여합니다. 양쪽 여백을 함께 설정하거나 각 측면을 개별적으로 설정할 수 있습니다.	<margin=10>
mark	지정한 색상으로 강조 표시합니다.	<mark=#FFFF0080>
mspace	문자의 폭을 문자 크기와 관계 없이 지정한 값으로 고정합니다.	<mspace=20>
nobr	태그 안의 문자들을 하나의 묶음으로 취급해 줄 바꿈합니다. 묶인 문자끼리는 최대한 줄 바꿈을 줄이게 됩니다. 입력 영역보다 문자가 길 경우 줄 바꿈 처리됩니다.	<nobr>
noparse	작성한 태그를 그대로 보여줍니다.	<noparse><i><rotate=50>보여질 문장
page	페이지를 나누어 보여줍니다. Overflow 프로퍼티가 Page로 설정되어 있어야 합니다.	첫 페이지<page>둘째 페이지
pos	태그 뒤의 문자가 입력한 값만큼 벌어져서 보입니다.	<pos=50>,<pos=30%>
rotate	입력한 글자를 기준으로 지정한 값만큼 회전시킵니다.	<rotate=45>
s	취소선입니다.	<s>
size	글자 크기를 조정합니다. 수가 클수록 크기가 커집니다.	<size=150%>
smallcaps	영어 알파벳 중 소문자를 작은 대문자로 표시합니다. 대문자는 변하지 않습니다.	<smallcaps>
space	설정한 값만큼 띄어쓰기를 합니다.	<space=50>
sprite	기본 이모지 또는 TextSpriteSet에서 정의한 이미지를 사용할 수 있습니다. 기본 이모지는 1부터 16까지의 값을 사용합니다.	<sprite=1>, <sprite name="my sprite">
strikethrough	취소선으로 표시합니다.	<strikethrough>
style	기본 스타일 또는 TextStyleSheet에서 정의한 스타일을 사용할 수 있습니다. 사용 가능한 기본 스타일은 다음과 같습니다.
* H1, H2, H3, C1, C2, C3, Quote, Link, Title, Normal	<style="H1">, <style="my style">
sub	아래 첨자로 변환합니다.	<sub>
sup	위 첨자로 변환합니다.	<sup>
u	밑줄을 긋습니다.	<u>
uppercase	영어 알파벳을 모두 대문자로 변환합니다.	<uppercase>
voffset	세로 오프셋을 설정합니다.	<voffset=10>,<voffset=10em>
width	가로 크기를 변경합니다.	<width=50%>
TextComponent 리치 텍스트 태그
TextComponent에서 사용 가능한 리치 텍스트 태그 목록입니다. IsRichText 프로퍼티를 활성화해야 리치 텍스트를 사용할 수 있습니다.

<태그이름 = 값> 글씨 입력 </태그 이름>

태그	설명	예시
b	글씨를 굵게 표현합니다.	<b>보여질 문장</b>
i	글씨를 기울어지게 표현합니다.	<i>보여질 문장</i>
size	글씨의 크기를 결정합니다.	<size=70>보여질 문장</size>

aqua, cyan, black, blue, brown, darkblue, fuchsia, green, grey, lightblue, lime, magenta, maroon, navy, olive, orange, purple, red, silver, teal, white, yellow
