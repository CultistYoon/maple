[Claude Code 설치 및 연결하기]


1. 클로드 설치되어 있는지 확인하기
claude --version

2. 클로드 설치하기
irm https://claude.ai/install.ps1 | iex

3. 환경변수 경로 추가하기
[Environment]::SetEnvironmentVariable("Path", [Environment]::GetEnvironmentVariable("Path","User") + ";$env:USERPROFILE\.local\bin", "User")

4. 설치후 클로드 확인하기(새 터미널에서 확인)
claude --version

5. 해당 경로로 이동하기
cd D:\MSW_AI

6. 해당 경로에서 클로드 실행
claude

7. 서버 연결
/mcp

8. 클로드 다시 실행
cd D:\MSW_AI
claude
